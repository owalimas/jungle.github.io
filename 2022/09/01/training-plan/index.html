22年本科项目培训计划

发表于2022-09-01|更新于2022-09-01|培训

项目培训涉及知识

写在前面：

有问题QQ联系：武俊（1204364750）、罗干（2507909599）、黄河（1353274029）、韩超（287811015）

该内容已同步到我个人博客，供随时维护与查看：

前端-自行补充

  1. JavaScripts基础部分，熟悉如何操作DOM元素，另外JS高级中的原型链、闭包等需要有所了解 【两周】
  2. jQuery jQuery 教程 | 菜鸟教程 (runoob.com) 熟练运动进行DOM操作 【一周】
  3. Bootstrap 前端框架，搞清楚其布局中涉及到的Class以及提供的一些特性 【一周】

前端-项目补充（项目培训包含）

  1. AvalonJs MVVM架构模型
  2. EasyUI 前端框架
  3. Freemarker 模板引擎
  4. Vue 部分项目涉及

后端-自行补充

  1. JavaSe基础知识【自行学习补充，很重要，两周时间】重点：反射，集合，面向对象，基本语言特性（建议尚硅谷：宋红康）

  2. JavaWeb 【自己完全没学过，很重要，建议补充，两周时间】 https://www.bilibili.com/video/BV12J411M7Sj?spm_id_from=333.880.my_history.page.click

  3. SSM框架 【很重要，两周时间】 https://www.bilibili.com/video/BV1Fi4y1S7ix?p=68&spm_id_from=333.880.my_history.page.click&vd_source=6f710bcdef4c9e851a9511462764faab

  4. 了解SpringBoot是什么？如何写配置文件，如何启动，做了哪些简化即可【一周时间】https://www.bilibili.com/video/BV1Lq4y1J77x?spm_id_from=333.337.search-card.all.click

  5. SpringBoot项目网上练习项目（找适合自己的，B站搜索：黑马和尚硅谷）【很重要两周时间】

  6. SpringCloud、微服务、Redis、RabbitMQ、ElaticSearch（项目组考核通过后，做项目后再去了解）

数据库-自行补充

  1. 通过mysql学会DDL和DML语句操作即可，需要了解索引这些知识【很重要 两周】

工具

  1. Git 学会使用Github/Gitee，本地搭建Git仓库然后进行自己的代码上传等操作【两天】
  2. Maven，从Jar包管理过度道Maven pom文件管理 【三天】
  3. Idea 常用快捷键要熟练
  4. Markdown 学会使用md写笔记，用xmind脑图做知识归纳

考核内容

  1. 前端：Js、JQuery的基础使用，常见的CSS关键字含义，常见的BootStrap概念

  2. 后端：Spring核心组件的功能（IOC、AOP）、SpringMVC涉及到的注解以及MVC模式如何解耦开发、Mybatis基本编写与特性考察

  3. 数据库：基本的建表、联表查询、索引创建以及增删改字段和记录

    注明：具体考核内容后面会详细列出，但是需要认真去学习，去跟着教程敲，做笔记回顾

写在最后

  1. 目前，就业环境不好，Java开发所需要掌握的东西也是越来越多，越来越深，所以一定要打牢基础，然后进行锻炼
  2. 建议学习视频（B站）
    • 学会怎么用：狂神
    • 了解怎么用，有所原理理解：尚硅谷（较长）、黑马（精练）
    • 源码理解和原理深入：马士兵、图灵
  3. 建议学习网站（偏向于面试前阅读，或闲暇深入底层时阅读）
    • 算法练习：labuladong 的算法小抄 :: labuladong的算法小抄 (gitee.io)
    • 面试八股：Java 面试指南 | JavaGuide
    • 全栈知识体系：BAT大厂面试题与全栈知识体系结合 | Java 全栈知识体系 (pdai.tech)
    • 深入计算机网络、操作系统、MySql：小林coding (xiaolincoding.com)
  4. 上面提供的时间，是建议学习时长，可以根据自身情况调整，建议顺序
    • JavaSE
    • JavaWeb、Js、JQuery、BootStrap
    • SSM、MySql
    • SpringBoot、网上项目练习
    • 公司开发流程、开发框架、业务等培训 【一个月，考核通过者】
  5. 马士兵、图灵全套课程网盘链接，学到后期找我拿！
